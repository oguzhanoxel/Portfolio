# Portfolio

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 12.0.0.

### Initial Setup

1. git clone https://github.com/oguzhanoxel/Portfolio.git
2. cd portfolio
3. npm install
4. ng serve
